This package contains models of proton exchange membrane fuel cells in the
Modelica language.  More information is available in the documentation, which
can be accessed through documentation layer of the Modelica code (starting from
package.mo), the help folder (starting from help/FCSys.html), or online at
http://kdavies4.github.com/FCSys.  In particular, see the "getting started"
section of the user's guide.
