FCSys.Conditions.ByConnector.Face.Single.Translational.PartialCondition

Partial function to select a translational quantity

Information

This function takes as inputs all the efforts and flows of the associated connector. It should be extended to add an algorithm that maps these inputs to a single value.

Extends from Modelica.Icons.Function (Icon for functions).

Modelica definition

partial function PartialCondition 
  "Partial function to select a translational quantity"
  extends Modelica.Icons.Function;

  // Material
  input Q.Density rho "Density (ρ)";
  input Q.Current Ndot "Diffusion current (Ṅ)";

  // Translational
  input Q.Velocity phi[Orient] "Velocity (φ)";
  input Q.Force mPhidot[Orient] 
    "Non-equilibrium force (mΦdot)";

  // Thermal
  input Q.TemperatureAbsolute T "Temperature";
  input Q.Power Qdot "Rate of thermal conduction (Q̇)";

  input Orient orient "Orientation of translational momentum w.r.t. the face";

  output Real x "Value of condition";
end PartialCondition;
