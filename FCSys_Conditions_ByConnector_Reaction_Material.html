FCSys.Conditions.ByConnector.Reaction.Material.PartialCondition

Partial function to select a material quantity

Information

This function takes as inputs all the efforts and flows of the associated connector. It should be extended to add an algorithm that maps these inputs to a single value.

Extends from Modelica.Icons.Function (Icon for functions).

Inputs

TypeNameDefaultDescription
CurrentNdot Rate of reaction (Ṅ) [N/T]
Potentialmu Electrochemical potential (μ) [l2.m/(N.T2)]
Velocityphi[:] Velocity (φ) [l/T]
ForcemPhidot[:] Force (mΦdot) [l.m/T2]
PotentialAbsolutesT Specific entropy-temperature product [l2.m/(N.T2)]
PowerQdot Rate of thermal advection (Q̇) [l2.m/T3]

Outputs

TypeNameDescription
RealxValue of condition

Modelica definition

partial function PartialCondition 
  "Partial function to select a material quantity"
  extends Modelica.Icons.Function;

  // Material diffusion
  input Q.Current Ndot "Rate of reaction (Ṅ)";
  input Q.Potential mu "Electrochemical potential (μ)";

  // Translational advection
  input Q.Velocity phi[:] "Velocity (φ)";
  input Q.Force mPhidot[:] "Force (mΦdot)";

  // Thermal advection
  input Q.PotentialAbsolute sT "Specific entropy-temperature product";
  input Q.Power Qdot "Rate of thermal advection (Q̇)";

  output Real x "Value of condition";
end PartialCondition;
