FCSys.Tests

Models and functions for test and validation

Information



This package may be safely removed from the
FCSys distribution (along with resources/NIST.xls), but it may be helpful for debugging.
The structure  of the subpackages matches that of FCSys,
although not all packages are represented.




The CallAll() function calls of the test
functions and the RunAll model includes all of the
test models.  Both should be tested to verify the FCSys
package.

FCSys.Tests.callAll

Call all of the test functions for FCSys


Information



  This function call will fail if any of the functions return an
  incorrect result.  It will return true if all of the functions pass.
  There are no inputs.

Extends from Modelica.Icons.Function (Icon for functions).

end callAll;
