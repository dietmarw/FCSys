FCSys.BaseClasses.Icons

Icons to annotate and represent classes

Information

Extends from Modelica.Icons.Package (Icon for standard packages).

Package Content

NameDescription
 BCs Icons for boundary conditions
 Blocks Icons for blocks (imperative or causal models)
 Names Icons labeled with the name of the class at various positions
 SignalBuses Icons for buses of signals (expandable connectors)
 Cell Icon for a cell
 PackageUnderConstruction Overlay that indicates a package is under construction
 Sensor Icon for a sensor
